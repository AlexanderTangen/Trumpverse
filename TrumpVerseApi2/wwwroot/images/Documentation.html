TrumpVerse API Documentation

Base URL

The base URL is: http://localhost:5259/api

All requests are made using GET, POST, PUT, and DELETE methods over HTTPS.

Available Resources

  • TrumpMerch: Retrieve or manage Trump merchandise.

GET /api/TrumpMerch

This request gets a list of all Trump merchandise in the store.

Example Response

[
  {
    "id": 1,
    "name": "Maga Hat",
    "price": 50,
    "description": "Iconic red hat with bold white 'Make America Great Again' embroidery.",
    "imageUrl": "/Images/Maga_Hat.jpg",
    "stock": 50
  }
]

POST /api/TrumpMerch

This request creates a new Trump merchandise item in the store.

Request Body (multipart/form-data)

{
  "name": "Maga Hat",
  "price": 50,
  "description": "Iconic red hat with bold white 'Make America Great Again' embroidery.",
  "image": "file (uploaded image)",
  "stock": 50
}

Example Response

{
  "id": 1,
  "name": "Maga Hat",
  "price": 50,
  "description": "Iconic red hat with bold white 'Make America Great Again' embroidery.",
  "imageUrl": "/Images/Maga_Hat.jpg",
  "stock": 50
}

PUT /api/TrumpMerch/{id}

This request updates a existing Trump merchandise item.

Request Body (multipart/form-data)

{
  "id": 1,
  "name": "Maga Hat",
  "price": 50,
  "description": "Iconic red hat with bold white 'Make America Great Again' embroidery.",
  "image": "file (uploaded image)",
  "stock": 60
}

After this, the item will be updated with the new stock quantity.

DELETE /api/TrumpMerch/{id}

This request removes a Trump merchandise item from the store.

Provide the id of the item in the URL to delete it.

Example Request

DELETE /api/TrumpMerch/1

Example Response

{
  "message": "Item deleted successfully."
}

POST /api/TrumpMerch/update-stock

This request updates the stock for multiple items at once.

Request Body (JSON)

[
  {
    "id": 1,
    "quantity": 10
  }
]

After this, the stock for the items will be updated.
